# MERN Authentication Learning Project

## Overview

A backend authentication project built while learning the MERN stack. This project focuses on implementing user management, database connectivity, and authentication concepts using Node.js, Express.js, and MongoDB.

## Features Implemented

* Express Server Setup
* MongoDB Atlas Connection
* Environment Variables using dotenv
* User Schema with Mongoose
* MVC Project Structure
* Signup Route and Controller Logic

## Tech Stack

* Node.js
* Express.js
* MongoDB Atlas
* Mongoose
* JWT
* bcryptjs
* Postman

## Project Structure

Backend/

* config/
* controllers/
* middleware/
* models/
* routes/
* utils/
* app.js
* server.js
